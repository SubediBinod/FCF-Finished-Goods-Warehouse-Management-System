# Finished Goods Warehouse Management System

A production-grade warehouse inventory and logistics management system developed for First Choice Foods Limited to manage finished goods storage, pallet tracking, dispatch operations, and warehouse analytics across multiple cold storage chambers.

---

## 🚀 Overview

This system digitizes warehouse operations including:

* Finished goods inward entry
* Pallet-based storage tracking
* Chamber-wise inventory management
* Dispatch and transfer workflows
* Audit logging and reporting

---

## 🏗️ System Architecture

* Frontend: React + Tailwind CSS + Recharts
* Backend: Spring Boot (REST APIs)
* Database: MySQL
* Authentication: JWT + Role-based Access Control
* Deployment: Ubuntu LTS (Internal Server)
* Remote Access: Tailscale VPN

---

## 📦 Key Features

### Warehouse Operations

* Product entry with pallet mapping
* Chamber-based storage system
* Location-based grid tracking (C1, C2, C3)
* Pallet transfer between chambers

### Dispatch System

* Manual dispatch tracking
* Customer-based dispatch records
* Stock deduction logic

### Analytics Dashboard

* Stock distribution charts
* Chamber occupancy visualization
* Product-wise inventory tracking
* Movement analytics (Recharts)

### Audit & Logs

* Inward logs
* Dispatch logs
* Transfer logs
* User activity tracking
